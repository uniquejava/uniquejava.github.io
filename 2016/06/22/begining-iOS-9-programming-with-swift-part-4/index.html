我才发现, 选中button然后点pin按钮, 弹出的constraints框中是离它最近的邻居的值(并不一定是superview)

创建Map view

  1. 拖一个view controller到storyboard
  2. 拖一个map kit view并设置为全屏(resize the map view to fit the view and then add missing constraints)
  3. 新建MapViewController继承自UIViewController设定为map view controller的custom class

  4. 给map view弄个outlet, 再创建一个代表选中restaurant的变量.

Detail view切到Map View并传参

Ctrl 拖动map button到新建的map view controller 选择segue的类型为Show, ID为showMap.
顺便把当前的restaurant做为参数通传过去.

Form 表单

  1. 拖一个table view controller到storyboard
  2. 从document outline中选中Table View将content属性从Dynamic Prototypes改为Static Cells.
  3. 此时table view会自动生成3个cell, 我们需要5个cell
  4. 从document outline中选中Table View Section, 修改rows=5

各个cell的作用如下:

Cell #1 - an image view for displaying the restaurant image
Cell #2 - a Name label and a text field
Cell #3 - a Type label and a text field
Cell #4 - a Location label and a text field 
Cell #5 - a Have You Been Here label and two buttons for Yes/No

Cell 1

  1. 选中第一个Tabel View Cell, height=250, bgcolor=light gray
  2. 拖一个image view到这个cell, image=photoalbum, 把它放在cell的正中间,然后add missing constraints
