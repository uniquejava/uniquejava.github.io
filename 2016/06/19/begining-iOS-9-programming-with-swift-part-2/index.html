使用Custom Cell (UI部分)!

basic style cell里边的thumbnail大小和位置都是固定的, 看起来丑爆了, 接下来我们要做以下几件事:
>

  1. 改变cell的高度
  2. 让thumbnail更大
  3. 显示更多restaurant的信息,比如location, type
  4. 设置字体类型和大小
  5. 显示圆角图片(而非矩形)

  1. 把Cell的style从basic改为custom
  2. 把table view的row height 和 cell的row height都改为80(呃..)
  3. 拖一个image view到cell里边, 位置为(14, 10, 60, 60)
  4. Label(name=Name, font=Text Styles - Headline)
  5. Label(name=Location, font style=Light, font-size=14, color=Dark Gray)
  6. Label(name=Type, font style=Light, font-size=13
  7. 使用stack view管理这里的image view和label, 省去给这些label和image view逐个加约束的麻烦.
  8. 按住cmd同时选中3个label, 点击stack按钮将它们装到vertical stack view, 设置spacing=1
  9. 同时选中image view和上一步中的stack view, 点击stack按钮将它们装到horizontal stack view, 设置spacing=10
  10. 设置上一步hsv自身的约束, 点击pin按钮,设定上右下左分别为(2,0,1.5,6), 这样stack view能充分利用整个cell的空间.
  11. 设定image view为固定大小(60x60), 方法是在document outline中ctrl水平拖动image view, 在弹出的菜单中shift选中width和height

segues

scene就是一屏, segue就是屏与屏之间的transition
In a storyboard, a scene usually refers to the on-screen content (e.g. a view controller). Segues sit between two scenes and represent the transition from one scene to another. Push and Modal are two common types of transition.

给view controller添加navagation bar的方法
选中view controller, 点击editor > embedded in > navigation controller, 然后点击view controller中的navbar部分, 设置它的title属性即可.

设置segue的方法
拖动cell到新的view controller上面, 选择一个segue的类型即可

4种类型的segues:
>

  1. show 新页面push到当前页面之上, 新页面会显示后退按钮
  2. show detail 新页面替换当前页面, 不会显示后退按钮
  3. present modally 从底部滑出并占满整屏(类似calendar点击+)
  4. present as popover (类似tooltip)

给segue命名
两个view controller之间很可能会有多个segue, 选择某个segue, 打开attributes inspector, 在identifier一栏给segue指定唯一标识符(e.g.: showRestaurantDetail)
