导航与Segue (ch 13)

Segue就是Scene与Scene之间的transition.

iOS中运行的应用拥有超过一个场景时，我们需要在不同场景之间进行切换，例如在主界面点击注册按钮即切换到到注册场景。场景之间的切换过程即为Segue。伴随着Segue的往往还会有数据交换。通过StoryBoard中的鼠标拖拽可以快速的建立场景间切换所需的Segue。

新建navigation controller

给view controller添加navagation bar的方法:

  1. 选中view controller, 然后Editor > Embedded in > Navigation controller
  2. 在当前的view controller顶部多出一个叫Navigation Item的东西, 这个就是navigation bar
  3. 可以选中它, 并在Attribute Inspector修改它的title属性

新建detail controller

  1. 拖一个view controller到storyboard
  2. 拖一个image view到view controller并设置image为全屏
  3. 修改image view的mode,从Scale to Fill改为Aspect Fill.

  4. 新建RestaurantDetailViewController继承自UIViewController并设置为view controller的custom class.

设置image全屏的方法

拉动image的左上和右下让它与superview边缘对齐, 点击pin按钮然后点击上右下左那4条bar让它们变成红色的实线最后Add 4 constraints即可

导航: 从list view到detail view

Ctrl拖动prototype cell到新的view controller, 在弹出的菜单中选择Selection Segue - Show

Segue的类型

4种类型的segues:
>

  1. show 新页面push到当前页面之上, 新页面会显示后退按钮
  2. show detail 新页面替换当前页面, 不会显示后退按钮
  3. present modally 从底部滑出并占满整屏(类似calendar点击+)
  4. present as popover (类似tooltip)

给segue命名

两个view controller之间很可能会有多个segue, 选择某个segue, 打开attributes inspector, 在identifier一栏给segue指定唯一标识符(e.g.: showRestaurantDetail)

从list view传参到detail view

prepareForSegue(:segue), UITableViewController的模板代码中已包含了这个方法, 只是默认注释掉了.

UIImageView的拉伸模式

UIImageView继承自UIView, 它有一个mode属性用来决定view如何包裹它的内容.

三种模式(page 236)
>
Scale to Fill(默认值, 填充View, 图片可能会被拉伸变形)
Aspect Fit(等比缩放, 但是图片的上下或左右可能出现空白)
Aspect Fill(等比缩放, 但是图片的某边可能超出边界)

最佳组合: Aspect Fill并且和With Clip Subview一起使用, 这个选项能裁掉多出的部分

什么是Dynamic Type

如果Cell能自适应大小(见上一节), 并且cell中label的字体是text style类型(比如Headline),而不是固定大小的font type, 这时这个label的字体大小就会随用户的设置动态改变.

用户设置字体大小的步骤: Settings > General > Accessibility > Larger Text or Settings > Display & Brightness > Text Size

Unwind Segue(关闭modal view)

在新的场景中完成对应任务后，需要切换回原来的场景，并将新场景中的数据传递回去，这个过程即为Unwind Segue。在StoryBoard中右击触发切换的按钮并拖拽到Scene Dock中的Exit按钮，即可选择对应的Unwind方法。当然在此之前需要在对应的view controller中重写 prepareForSegue(_:)方法并对需要传递的数据做好准备，并实现具体的Unwind方法。

从modal返回前一屏需要定义unwind segue, 步骤是

  1. 在detail view中定义一个unwind回调方法(方法名随意)

    @IBAction func returnToMe(segue:UIStoryboardSegue) {}

  2. 在modal view中ctrl拖动关闭按钮到Scene dock的Exit图标上, 然后在弹出菜单中选择先前定义的returnToMe:方法

Animation概述

动画包含一个start state和一个end state

  1. 在viewDidLoad中设定初始状态:view.transform = start_state
  2. 在animate的回调函数中设定结束状态:view.transform = end_state
  3. 在viewDidAppear中开始动画:
    UIView.animateWithDuration(duration:delay:options:animations:completion:)

注: animateWithDuration是UIView的静态方法.

处理表情按钮的点击事件.
