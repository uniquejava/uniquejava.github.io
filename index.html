5 使用Auto Layout

前面章节将Size classes去掉后各类组件是用的固定定位, 别看Hello World按钮在canvas中是居中的,实际上这个按钮是用相对于左上角的偏移量来定位的, 当Cmd + Arrow Left左旋或Cmd + Arrow Right右旋(切换到横屏模式),或者是切换到其它尺寸的iPhone, Hello World按钮就不再居中.

为什么用points而不是pixels

刚推出iPhone的时候屏幕大小为3.5寸 (320x480像素) 那个时候, 1 point = 1 pixel.
后来iPhone4出现了Retina屏 (640*960像素), 1 point = 2 pixes的, 但在设计的时候还是用point来表示, 这样不论是不是Retina屏, 用同一个值就能搞定不同的分辨率.

基准的分辨率
>
iPhone4/4S: 320x480
iPhone5/5S: 320x568
iPhone6/6S: 375x667

即使屏幕变成640x960或1280x1920, 也只不过是1point变成了2pixel或4pixel, 我们还是用point来做设计,而将单位转换交给iOS系统来处理

公式 button.left = (container.left + 30)

auto layout bar

Each button has its own function:

  1. Stack – Embed views into a stack view. Stack view is a new feature in Xcode 7.
  2. Align – Create alignment constraints, such as aligning the left edges of two views.
  3. Pin – Create spacing constraints, such as defining the width of a UI control.
  4. Issues – Resolve layout issues.

实例一(居中对齐)

比如要居中对齐按钮, 那么先选中按钮, 然后点Align 然后在弹出的popover中选择Vertical center in container and Horizontal center in container 最后点击Add 2 Constraints.

当没有layout issue的时候, constraint line是蓝色的, 此时在canvas中将按钮挪动位置, constraint line会变成橙色, 警告你位置不匹配.

此时可以选中这个按钮, layout bar中点Issues按钮然后选择Update Frames就可以了. 如果要重新给这个按钮添加contraints, 可以点击Clear Constraints

预览

在storyboard左上角点有4个小方格的图标 > Preview(1) > 按住Option键并点击Main.storyboard(Preview), 在打开的界面中点击+号来选择不同尺寸的设备.

实例二 (ctrl + drag label 右0 下20)

比如要给label设定右边距0, 下边距20, 这个边距叫spacing constraints, 在IB中, 可以ctrl drag这个label到它自身, 或是其它参照物来添加constraints.

  1. ctrl drag label向右, 鼠标划过的参照组件会高亮成蓝色, 如果是label本身, 可以设置width, 如果是container view, 可以设定trailing space 和 center vertically
  2. ctrl drag label向左, 设定width或leading space和center vertically
  3. ctrl drag 向上, 设定height或vertical spacing to top 或 center horizontal
  4. ctrl drag 向下, 设定height或vertical spacing to bottom和 center horizontal

注意:

  1. 设定边距的时候不一定要拉到参照物的边缘上. 只要浮到参照物上(高亮成蓝色)就可以.
  2. 无论左右, 拖到自身可以设定width, 拖到superview可以设定center vertically
  3. 无论上下, 拖到自身可以设定height, 拖到superview可以设定center horizontally

实例三(label 上40, 左右居中)

拖个label, ctrl + drag向上, 按住shift, 依次选择vertical spacing to top, center horizontally 和 add contraints 然后在size inspector中修改spacing的值为40, 最后Resolve Issues > Update Frames(快捷键cmd + option + =).
